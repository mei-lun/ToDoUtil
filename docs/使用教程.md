# ToDoUtil 使用教程

一个钉在桌面的小工具，帮你记住"今天要做什么"。本教程从零开始，跟着做就会用。

---

## 它长什么样

打开后，你会看到一个窄窄的小窗口，竖着的一条，默认贴在桌面右上角左右。它会一直浮在所有窗口前面（你写邮件、刷网页时它都看得到），所以不用担心被挡住。

窗口里从上到下：

- 最上面：一个加号 ＋
- 中间一栏：日期切换条 `◀ 今天 06/23 ▶`，右边还有两个图标 📥 和 📚
- 下面：你写下的任务

---

## 第一步：记下一件事

**点左上角的 ＋** → 它会变成一个输入框 → 输入你要做的事，按回车即可。

例如输入"买牛奶"按回车，下面就会出现一行：

```
○ 买牛奶
```

那个空心的圆圈是"还没做"。

> 不想记了？直接按 `Esc`，或在输入框里啥也不打就点别处，它会自己缩回 ＋。

---

## 第二步：做完了打勾

**点圆圈** → 圆圈变实心 ●，过 2 秒这条任务就消失了（它被存到"归档"里，后面会讲）。

万一手抖点错了？**别慌，2 秒内再点一次**那个实心的 ●，就能撤回。超过 2 秒才发现点错也没事——后面有"归档"可以查到，不过任务不会自动跑回来。

---

## 第三步：明天才做的事

如果你想说"这件事**明天**才做"，输入的时候在前面加 `@明天` 即可：

```
@明天 给妈妈打电话
```

按回车后，这条任务**今天看不到**，明天打开应用才会出现。

支持的写法（都要加 `@`）：

| 你想说的 | 这样写 |
|---|---|
| 今天 | `@今天` 或啥也不写（默认就是今天） |
| 明天 | `@明天` |
| 后天 / 大后天 | `@后天` / `@大后天` |
| 三天后 | `@三天后` 或 `@3天后` |
| 6 月 25 号 | `@6月25日` 或 `@6.25` 或 `@6-25` 或 `@6/25` |
| 周三 | `@周三` |
| 下周一 | `@下周一` |
| 上午 9 点 | `@上午9点` 或 `@9点` 或 `@9:00` |
| 下午 3 点半 | `@下午3点半` 或 `@15:30` |

也可以日期 + 时间一起写：

```
@明天9点 跟同事开会
```

写错了？没关系，软件不认识它就当成普通文字保留下来，不会乱归类。例如 `@xyz 啥` 会被当成今天的"@xyz 啥"。

---

## 第四步：在任务里加详细内容

每条任务除了标题，还能写**详细说明**——比如开会的议程、购物清单、参考链接、贴一张图。

**点任务那一行的空白处**（不是圆圈），它会展开一块详情区。

第一次展开是空的，写"（无内容）"。**点右上角的铅笔 ✎** 进入编辑模式，就能输入文字了：

- 想换行就直接换行
- 想列清单就用 `- 项目1` `- 项目2`
- 想加可勾选的小项目，用 `- [ ] 第一步` `- [ ] 第二步`（注意空格）
- 想贴图片：直接 **Ctrl+V 粘贴** 或者 **从文件夹拖** 进来都可以
- 写超链接：直接粘贴网址即可
- 加粗某段文字：用 `**粗体**`

写完点别处自动保存，或者按 `Ctrl+S` 保存。要放弃修改按 `Esc`。

**小清单（子任务）的妙用**：如果你写了 `- [ ] 第一步` 类的小项目，保存后那一行的方框可以直接点击勾选。每勾完一个，主任务那行右边会显示进度，比如 `2/5` 表示一共 5 个小步骤已完成 2 个。主任务什么时候算完成由你自己决定（手动点圆圈），不会因为小项目都勾完就自动消失。

---

## 第五步：今天没做完怎么办

打工人日常：列了 5 件事，下班前还有 2 件没做。三种处置方法，鼠标移到那条任务上时，**右边会浮出 4 个小图标**：

```
→ 📥 ✎ ✕
```

它们的意思是：

- **→（推迟一天）**：明天再做。任务挪到明天的列表里。
- **📥（搁置）**：暂时不想追这件事，先放到一边。**不会出现在任何日期的列表里**，需要时再拿出来。
- **✎（编辑详情）**：进入详情编辑（跟点铅笔一样）
- **✕（放弃）**：彻底不做了。会问你确认一下。

**还有个偷懒模式**：如果你啥都不操作，到了第二天 0 点，软件会**自动**把今天没做完的任务挪到第二天。你早上打开就能直接看到。

---

## 第六步：延期了多久看得到

如果一件事一直拖着没做（比如你"自动滚动"了好几天），任务行的左边会出现一小块**红色方块**。

- 浅红：拖了 1-2 天
- 深一点的红：3-5 天
- 更深：6-10 天
- 暗红：超过 10 天

红得越深，你越该面对它了。

---

## 第七步：看其他日期的任务

**顶部的 `◀` 和 `▶`** 切换到昨天/明天/后天的列表。所有用 `@明天` 创建的任务都在那边等着。

---

## 第八步：搁置池——"以后再说"

刚才说的"📥 搁置"，被搁置的任务存到了**搁置池**里。它们没有日期，不会打扰你，但也不会自己消失。

**点顶部的 📥 图标** → 进入搁置池视图，能看到所有暂时搁置的事。

需要重新捡起来某件事？点它右边的 **"拉回到…"** 按钮，弹出一个小日历选哪天做，确认后任务就回到那天的列表里了。

什么时候适合搁置：

- 你确实想做，但当下不是合适的时机（"等天气好了再去公园"）
- 想了一天没头绪，先不追了
- 是个长期 idea，不知道哪天会启动

---

## 第九步：归档——回顾你的成就

每次你勾完任务（圆圈 → 实心），过 2 秒它就进了**归档**。

**点顶部的 📚 图标** → 进入归档视图。

里面按日期倒序展示你完成过的所有事。每条右边有一根**长长的彩色条** + 一个数字，表示这件事**从计划到完成花了多少天**：

- 当天就搞定的事 → 短短一格 + "1 天"
- 拖了 10 天才完成 → 长长一条 + "10 天"
- 放弃的事 → 灰色的条 + ~~删除线标题~~（依然记录，提醒你"曾经想做但放下了"）

可视化地告诉你：你最近忙在哪些事上、哪些事拖太久了。

**两种视图切换**（顶部按钮）：

- **时间线**（默认）：竖着列出每一天完成了啥
- **网格**：像 GitHub 那种贡献图，横着是时间，每条任务一行小方块。点这个的时候窗口会临时变宽，回到时间线时会自动缩回原来的宽度。

**搜索**：归档顶部有个搜索框，输入关键词（标题或详情里的文字都行）就能快速找到几个月前完成的某件事。

---

## 第十步：让窗口跟你"互动"

### 移动窗口

窗口没有标题栏（为了好看简洁），所以普通的拖动方式不太一样：

**方法 1**（最常用）：点最上面那条空白条（在 ＋ 的空白处），按住可以直接拖。

**方法 2**（移动模式）：右键托盘图标 → 选 **"↔ 移动窗口"** → 整个窗口变浅蓝色 → 这时鼠标按住任意位置都能拖，松手后自动退出。

### 永远在最前面 / 取消

右键托盘图标 → 点 **"✓ 钉在桌面"** 切换。打勾就是钉住，再点一下取消。

### 关掉窗口（但不退出）

按 `Esc`（在没有别的输入/编辑时）窗口会"消失"。其实只是收回到了托盘——**左键点托盘图标**就能让它再出现。

### 想完全退出

右键托盘 → **退出**。

### 让窗口大小自动调整

窗口默认很小（300×360），随着你的任务变多、详情打开，**窗口会自动变高**。但不会自己变小（避免忽然跳来跳去）。

如果觉得太大想缩，鼠标移到窗口底边出现可拖的边线，往上拖即可。

---

## 第十一步：在任意地方记一笔

经常这样：你正在浏览器看东西，突然想起"晚上要订机票"——不用切到 ToDoUtil 主窗口，**按 `Ctrl+Alt+T`** 屏幕中央会弹出一个小输入框，输入"@今晚 订机票"按回车，自动关闭。

也可以右键托盘图标 → "+ 快速添加任务"。

---

## 第十二步：调整快捷键、改保存位置等

右键托盘 → **设置**：

- **启动时总在最前**：勾上就开机一直钉在桌面
- **全局添加任务快捷键**：默认 `Ctrl+Alt+T`，嫌它跟别的软件冲突可以改。点输入框，按下你想要的组合键即可（必须有至少一个普通键，不能只是 Ctrl+Alt）。改完点"保存并重启"。
- **数据目录**：你的所有任务都存在哪个文件夹。如果你想让两台电脑同步，把这里改成 OneDrive / 坚果云之类的同步文件夹即可。点"打开目录"可以直接看到那些文件。
- **保留备份天数**：每天软件都会自动备份一次（防止文件损坏），默认保留 30 天的备份。

---

## 第十三步：手动调任务顺序

某条任务想往上挪一点（比如更紧急）：

**长按那一行 0.3 秒**（不是点击，是按住不放），整行会变蓝色 → **拖到你想要的位置** → 松手。顺序就调整好了。

要避免误触，所以设计成"按一下就开始拖"是不行的，必须按住一会儿。

## 常见问题

**Q：我不小心关掉了，任务还在吗？**
A：在。叉掉只是收到托盘，没有真正退出。即使你"退出"了，下次打开任务也都还在——它们存在你电脑的文件里，不会丢。

**Q：电脑死机/突然断电会丢数据吗？**
A：基本不会。每次保存都用了"先写临时文件再替换"的方式，避免半截写入损坏。即使真出了问题，每天首次启动时会自动备份一次，最近 30 天都能恢复——位置在设置 → 数据目录 → backups 文件夹。

**Q：图片加多了，电脑会不会撑爆？**
A：图片存在数据目录的 attachments 文件夹下。每张图按内容去重（同一张图粘贴 5 次也只占 1 次空间）。但归档很久的任务里的图也会一直保留——长期使用后那个文件夹会变大，不影响使用。

**Q：搁置的任务我忘了怎么办？**
A：顶部那个 📥 图标会显示一个小数字，告诉你池子里还有几件事待你处理。

**Q：归档后能改任务内容吗？**
A：不能。归档是历史记录，只能看不能改——这是故意的，避免你回头修改记录。

**Q：怎么换电脑？**
A：把数据目录里所有文件复制到新电脑同名位置即可。或者在两台电脑设置里把数据目录指向同一个云盘文件夹（OneDrive / iCloud / 坚果云之类），就是一份数据多端用。**注意：两台电脑不要同时打开**，会冲突。

## 一个典型的一天

早上 9 点，打开电脑，ToDoUtil 已经在桌面右侧。看到：

```
○ ▓ 写周报         （延期 1 天，浅红块）
○   买咖啡 09:30
○   开会  10:00
○   @明天 ... 不在今天显示
```

9:30 顺手点掉"买咖啡"。继续工作，写完周报点掉。开会前 5 分钟想起还要带笔记本——按 `Ctrl+Alt+T` 弹出快速输入，敲"带笔记本"回车，关闭。

下午写文档思路卡住了，点击那条"写文档"任务展开详情，写下三个备选方案，回头再选。

下班前还有一条"健身"没做完，鼠标移上去 → 点 **→** 推迟到明天。

周日翻 📚 归档看本周完成了什么，看到几条线特别长（拖了一周才完成）——下周开始用搁置池而不是任由它们累积。

---

完。有任何不顺手的地方都是软件该改的地方，记下来反馈即可。
